프로그램

for Windows

llpv (llProcessView)

활성화된 프로그램의 사용시간을 측정하여 컴퓨터를 생산적으로 사용할 수 있도록 도와주는 프로그램입니다.

현재는 Windows에서 java 프로그램으로 동작하며 jna 라이브러리를 사용하여 활성화된 프로그램을 가져오고 있습니다. 향후 계획은 여러 Windows 뿐만아니라 운영체제 지원과 데이터를 기반으로 다양한 기능을 구현할 예정입니다.

Get Started

기능

순위, 검색, 제어, 알람, 메모 기능이 있고 홈페이지에서 사용자들의 실시간 프로세스 순위를 볼 수 있습니다.

순위

프로세스명 또는 프로그램 제목 별 사용시간과 순위를 알려줍니다.

검색

프로세스명 또는 프로그램 제목으로 데이터를 검색할 수 있습니다. 검색 조건은 대소문자 구분없이 두가지 필드를 검색하며 한 글자라도 포함되면 검색합니다.

제한

프로그램 제한시간을 설정할 수 있습니다. 제한시간이 되기전에 알림을 주고 제한시간을 넘어가면 해당 프로그램이 강제 종료됩니다.

알람

지정한 알림 시간만큼 컴퓨터를 사용하면 윈도우 화면잠금이 되고 휴식시간이 지나면 트레이 알림이 옵니다.

메모

간단한 메모 기능입니다. 한줄 한줄 작성할 때마다 왼쪽 공간에 시간이 기록됩니다. (자동저장 시간 10초)
